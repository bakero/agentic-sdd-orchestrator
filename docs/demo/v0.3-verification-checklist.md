# v0.3 Verification Checklist

Use this checklist to verify the v0.3 project manager and doctor release. Run from the orchestrator repository unless noted otherwise.

## Automated checks

- [ ] `npm test` passes.
- [ ] `npm run typecheck` passes.

## Project manager

- [ ] `npm run agentic-sdd -- project add <sandbox> --name <name>` works and reports a recommended next command.
- [ ] `npm run agentic-sdd -- project list` shows the registered project (name and path).
- [ ] `npm run agentic-sdd -- project inspect <name>` works and reports git/runtime/package/feature status.

## Doctor and next: setup phase

- [ ] `npm run agentic-sdd -- doctor <name>` detects the missing runtime kit (`NEEDS_SETUP`, runtime folders/workflow/scripts reported as `FAIL`).
- [ ] `npm run agentic-sdd -- next <name>` recommends `install`.

## Install

- [ ] `npm run agentic-sdd -- install <sandbox>` works and reports copied files / missing scripts as `none`.
- [ ] `npm run agentic-sdd -- doctor <name>` now reports runtime kit folders, guard workflow, and agent scripts as `PASS`.

## Doctor and next: feature phase

- [ ] `npm run agentic-sdd -- doctor <name>` detects the missing feature (`NEEDS_FEATURE`).
- [ ] `npm run agentic-sdd -- next <name>` recommends `init-feature`.

## init-feature

- [ ] `npm run agentic-sdd -- init-feature <sandbox> --issue 1 --slug demo-feature --title "Demo feature"` works.
- [ ] `npm run agentic-sdd -- doctor <name>` now reports the feature folder and its `status.md` as `PASS`.

## Doctor and next: dependencies and prompt phase

- [ ] `npm run agentic-sdd -- doctor <name>` detects missing dependencies and/or the missing generated prompt (`NEEDS_NEXT_PROMPT` or `CHECK_WARNINGS`, `node_modules`/`next_prompt.md` reported as `WARN`).
- [ ] `npm install` works inside the sandbox target repository.
- [ ] `npm run agent:next` works inside the sandbox target repository.
- [ ] `.agent_runtime/next_prompt.md` is generated.

## Final state

- [ ] `npm run agentic-sdd -- doctor <name>` reports `READY` (or `CHECK_WARNINGS` with only non-blocking warnings).
- [ ] `npm run agentic-sdd -- next <name>` recommends opening/copying `.agent_runtime/next_prompt.md`.

## Safety guarantees observed

- [ ] No autonomous agent execution occurred at any point.
- [ ] No external AI API call occurred at any point.
- [ ] No automatic merge or PR creation occurred at any point.
- [ ] `doctor` and `next` did not write to, or otherwise modify, the target repository.
- [ ] `project remove` (if exercised) did not delete or modify any target repository files.

## Backwards compatibility

- [ ] `npm run agentic-sdd -- inspect <target-repo>` still works unchanged.
- [ ] `npm run agentic-sdd -- install <target-repo> --dry-run` still works unchanged.
- [ ] Existing v0.1/v0.2 tests (`installer.test.ts`, pre-existing `index.test.ts` cases) still pass.
