# v0.1 Demo Script

## Goal

Show the local Cowork-mode MVP from a fresh sandbox to the first generated prompt.

## Preconditions

- You are in `C:\Users\Asus\agentic-sdd-orchestrator`
- `npm install` has been run in the orchestrator repo
- Node.js, npm, git, and PowerShell are available

## PowerShell flow

```powershell
$sandbox = Join-Path $env:TEMP "agentic-sdd-demo-sandbox"
if (Test-Path $sandbox) { Remove-Item -Recurse -Force $sandbox }
New-Item -ItemType Directory -Path $sandbox | Out-Null

git -C $sandbox init
git -C $sandbox config user.name "Demo User"
git -C $sandbox config user.email "demo@example.com"

@'
{
  "name": "agentic-sdd-demo-sandbox",
  "version": "1.0.0",
  "private": true
}
'@ | Set-Content -Path (Join-Path $sandbox "package.json")

npx tsx packages/cli/src/index.ts inspect $sandbox
npx tsx packages/cli/src/index.ts install $sandbox
npx tsx packages/cli/src/index.ts init-feature $sandbox --issue 1 --slug demo-feature --title "Demo feature"

Set-Location $sandbox

npm install
npm run agent:next
```

## Expected generated files

After `npm run agent:next`, these files should exist:

```powershell
Get-ChildItem .agent_runtime
```

Expected entries:

- `validation_report.json`
- `next_action.json`
- `context_files.txt`
- `next_prompt.md`

## Open the generated prompt

```powershell
Get-Content .agent_runtime\next_prompt.md
```

Or open it in your editor:

```powershell
code .agent_runtime\next_prompt.md
```

## Demo narration guidance

As you walk through the flow, emphasize:

- the orchestrator installs repository-local workflow assets, not hosted services;
- `init-feature` creates the first feature folder and branch scaffold;
- `npm run agent:next` does not execute an AI model;
- the output is a prompt for a human-supervised coworking tool;
- human final merge remains required.
